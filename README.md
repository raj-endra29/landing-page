# Hari Om Creations Landing Page

This is a landing page for Hari Om Creations, built to match the exact Figma design specifications.

## Files Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling matching the Figma design
- `images/` - Directory containing all images from Figma:
  - `logo.svg` - Company logo
  - `hero-background.png` - Hero section background image
  - `client-logos.svg` - Client logos section
  - `project-fairmont.png` - Fairmont project image
  - `project-jw-marriott.png` - JW Marriott project image
  - `project-park-hyatt.png` - Park Hyatt project image

## Design Specifications

### Colors
- Primary Background: `#F8F7F3`
- Navigation Background: `#243759`
- Accent Color: `#BC9B2F`
- Text Primary: `#1A1A1A`
- Text Secondary: `#666666`
- Dark Background: `#131F33`
- Button Color: `#8B741F`
- White: `#FFFFFF`

### Typography
- Primary Font: Avenir (with fallbacks)
- Secondary Font: Manrope (from Google Fonts)

### Layout
- Max Width: 1440px
- Responsive design for mobile, tablet, and desktop
- Exact spacing and padding as per Figma specifications

## Sections

1. **Navigation** - Sticky navigation with logo and menu items
2. **Hero Section** - Full-width hero with background image and CTA
3. **Client Section** - Trusted brands showcase
4. **Why Choose Us** - 4 feature cards in a 2x2 grid
5. **Featured Projects** - 3 project cards in a row
6. **Feedback Section** - Client testimonial
7. **View All Projects** - CTA section
8. **Footer** - Company info, links, and contact details

## How to Use

Simply open `index.html` in a web browser to view the landing page. All images and styles are included in the project directory.

- Firefox (latest)
